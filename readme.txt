=========================
    HOW TO USE IT?
=========================

Simply install the plugin on the client site, after activation look on the left hand side at the very top in the Dashboard and click on the Dashboard menu "Plugin Update Report" and choose the dates you with to see the updates for. If you are happy with the updates you can then download the data to a PDF file.
